# Student Council Election System

A web-based voting system for student council elections with Django backend and localStorage fallback.

## Features

- ✅ **Multi-position voting** (President, Secretary General, Education Secretary)
- ✅ **Voter authentication** using admission numbers (NUU001-NUU005)
- ✅ **Department tracking** for voting analytics
- ✅ **Real-time statistics** dashboard
- ✅ **Admin panel** for election management
- ✅ **Dual storage** - Django database + localStorage backup
- ✅ **Results export** to text file
- ✅ **Mobile-responsive design**

## Tech Stack

- **Backend**: Django 4.x
- **Frontend**: HTML5, CSS3, JavaScript
- **Database**: SQLite (default) / PostgreSQL (production)
- **Storage**: LocalStorage fallback
